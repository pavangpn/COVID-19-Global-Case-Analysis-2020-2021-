## 🦠 COVID-19 Global Case Analysis (2020-2021)
>>  End to end analysis of COVID-19 data with visualizations, correlations and insights.

## 🔹 Introduction
>>  COVID-19 emerged as a global health crisis in early 2020,affecting millions of people worldwide.
>>  Understanding the progression of confirmed cases,recoveries and deaths across countries is crucial for analyzing the 
    impact of the pandemic and identifying patterns in its spread.
>>  This project focuses on the data containing confirmed cases,recoveries and deaths from almost all countries between 
    "22 January 2020" and "29 May 2021".

## 🎯 Objective
    To perform Exploratory Data Analysis (EDA) using Python to clean the data and uncover trends,visualize the progression 
    of the pandemic and highlight differences across regions.

## 🛠️ Tools & Libraries
    Python
    Pandas
    Numpy
    Matplotlib
    Seaborn
    Jupyter Notebook
